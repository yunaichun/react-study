## 简介

此仓库主要是记录下自己学习 React、Redux 相关库源码的笔记和注释，方便以后忘记可以从学习轨迹中迅速上手。

## 目录结构

- react-src
- react-simple-diy1
- react-simple-diy2
- redux-src
- redux-simple-diy
- react-redux-src
- redux-promise-src
- redux-thunk-src
